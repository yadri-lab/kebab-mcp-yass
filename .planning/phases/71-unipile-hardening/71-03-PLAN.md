        `GET /api/admin/audit/unipile?account_id=&since=&tool=&result=&limit=&cursor=` returns paginated audit rows for the current tenant. Admin auth required. Cursor is base64 of last page's terminal `audit_id`. Default limit 50, max 200. Companion endpoint `GET /api/admin/metrics/unipile-quotas?account_id=&tool=` returns current quota usage per account/tool; `GET /api/admin/metrics/unipile-quotas/summary` returns the active-bucket matrix across all accounts × tools in the tenant.
